# motion-forecasting.github.io

Project page for [https://motion-forecasting.github.io/](https://motion-forecasting.github.io/), built from the [Nerfies](https://github.com/nerfies/nerfies.github.io) template.

## GitHub Pages

In the repo **Settings → Pages**, set the source to the branch that contains this site (e.g. **main**) and **/ (root)** so `index.html` is served.

## Customize

- **`index.html`** — title, authors, links (Paper / arXiv / Video currently point to `#`; set real URLs), YouTube embed (`VIDEO_ID`), abstract, sections, BibTeX.
- **`static/videos/`** — add `teaser.mp4`, `result-1.mp4` … `result-4.mp4`, `extra.mp4` (or rename files and update `<source>` paths).
- **`static/images/`** — optional favicon swap (`favicon.svg`).

Large videos are often tracked with [Git LFS](https://git-lfs.com/) or hosted externally; otherwise the repo can grow quickly.
